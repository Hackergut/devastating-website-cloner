# 📸 Screenshots Setup

To add professional screenshots to your README:

## Required Images

Place these images in `assets/screenshots/`:

1. **ledger-clone-demo.png** - Before/after comparison
   - Size: 1280x720 or larger
   - Show original website vs cloned version
   - Side by side comparison

2. **design-system-demo.png** - Design system extraction
   - Show extracted colors, fonts, spacing
   - Example: `docs/design-system/design-system.json`

3. **tailwind-config-demo.png** - Auto-generated config
   - Show `tailwind.config.js` with extracted values
   - Highlight the auto-generated colors/spacings

4. **components-demo.png** - Generated React components
   - Show component code with exact CSS values
   - Example: `src/components/sections/Header.tsx`

## How to Create Screenshots

### Method 1: Manual Screenshots

```bash
# Take screenshots manually
# 1. Original website in browser (1440px width)
# 2. Cloned website in browser (1440px width)
# 3. Place side by side in image editor
# 4. Save as ledger-clone-demo.png
```

### Method 2: Automated Screenshots

```bash
# Screenshots are automatically created in:
ls docs/extraction/screenshots/

# Copy to assets:
cp docs/extraction/screenshots/desktop.png assets/screenshots/ledger-clone-demo.png
```

### Method 3: Use Figmas/Sketch

Create professional mockups showing:
- Original site
- Cloned site
- Design system panel
- Generated code
